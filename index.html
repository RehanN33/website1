Sering Ditanyakan (FAQ)

Punya pertanyaan? Mungkin jawabannya ada di bawah ini.

🔒 Apakah data instansi tempat saya PKL aman?

Aman 100%. Kami menjamin kerahasiaan nama kamu, nama guru pembimbing, serta nama perusahaan tempat kamu prakerin.

🛠️ Kalau draf laporannya disuruh revisi bagaimana?

Kami memberikan garansi revisi gratis sesuai dengan catatan atau umpan balik (*feedback*) dari guru pembimbing sekolah kamu.

⏱️ Berapa lama proses pengerjaannya?

Normalnya 3-5 hari kerja tergantung tingkat kesulitan. Untuk kebutuhan mendesak, kamu bisa mengambil layanan kilat (hubungi admin).

📄 Format file apa saja yang nanti saya terima?

Kamu bakal terima file master mentah format Microsoft Word (`.docx`) yang bisa diedit lagi, file PDF, serta `.pptx` kalau pesan paket presentasi.
